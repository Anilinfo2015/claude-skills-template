# Text Summarizer

Intelligently condense text while preserving key information and meaning.

## Overview

This skill provides structured guidance for creating high-quality text summaries of varying lengths. Whether you need a brief overview or a comprehensive abstract, this skill ensures consistency and accuracy in summarization.

## Key Features

- Multiple summary length options (brief, standard, extended, custom)
- Preserves main ideas and key information
- Maintains original tone and intent
- Clear step-by-step process
- Examples for different content types
- Best practices and common pitfalls

## When to Use

- Creating executive summaries
- Generating abstracts
- Condensing meeting notes
- Writing TL;DR versions
- Reducing lengthy reports
- Summarizing research findings

## Quick Examples

**Long Email → Brief Summary:**
```
Input: 400-word email about project updates
Output: "Project on track for Q3 delivery. Budget approved. 
Hiring complete. Next milestone: beta testing in August."
```

**Research Paper → Abstract:**
```
Input: 5000-word research paper
Output: 150-word abstract covering background, methods, 
results, and conclusions
```

**Meeting → Action Items:**
```
Input: 60-minute meeting transcript
Output: Key decisions, action items, and next steps in 
100 words
```

## Summary Lengths

### Brief (2-3 sentences, ~50 words)
Perfect for: Email summaries, quick updates, social media
Focus: Main point + one key detail

### Standard (1 paragraph, ~125 words)
Perfect for: Executive summaries, abstracts, overviews
Focus: Main point + 2-3 supporting ideas + conclusion

### Extended (2-3 paragraphs, ~250 words)
Perfect for: Detailed summaries, comprehensive overviews
Focus: Full context + all major points + implications

## Content Types Supported

- Research papers and academic articles
- News articles and press releases
- Business reports and presentations
- Technical documentation
- Meeting notes and transcripts
- Email threads
- Book chapters and long-form content

## Quality Guarantees

✓ Accurate representation of original content  
✓ Preservation of key information  
✓ No added opinions or interpretations  
✓ Grammatically correct  
✓ Can stand alone without original  
✓ Maintains original tone

## Best Use Cases

### Executive Summaries
Busy executives need quick understanding of detailed reports. This skill creates summaries that capture essential information for decision-making.

### Academic Abstracts
Researchers need concise paper summaries. This skill follows academic conventions for abstract writing.

### Content Marketing
Marketers need engaging TL;DR versions. This skill maintains key messages while reducing length.

### Meeting Minutes
Teams need actionable summary of discussions. This skill extracts decisions and action items.

## Limitations

- Cannot add information not in original
- Very brief summaries may lose nuance
- Highly technical content may need specialized knowledge
- Does not replace reading for deep understanding

## Getting Started

1. Provide the text to summarize
2. Specify desired length (brief/standard/extended/custom)
3. Indicate target audience if relevant
4. Review generated summary
5. Request adjustments if needed

See [SKILL.md](SKILL.md) for detailed instructions and [examples.md](examples.md) for more examples.

## Tips for Best Results

- Provide clean, well-formatted input text
- Specify if technical terms must be preserved
- Indicate if specific information must be included
- Mention target audience for tone adjustment
- Request specific format if needed (bullet points, paragraph, etc.)
